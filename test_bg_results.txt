BG COMMAND IMPLEMENTATION - PROMPT 3.5
========================================

IMPLEMENTATION COMPLETE ✓

Features Implemented:
---------------------
1. Job Selection:
   - bg (no args): Resume most recent stopped job
   - bg %n: Resume specific stopped job n
   - bg n: Also accepts without % prefix

2. Job Status Handling:
   - Sends SIGCONT to stopped jobs to resume them
   - Updates job status from STOPPED to RUNNING
   - Marks job as background (background = 1)
   - Prints confirmation: [N]+ command &

3. Error Handling:
   - "bg: no current job" when no jobs exist
   - "bg: no stopped jobs" when no stopped jobs available
   - "bg: invalid job id" for malformed job IDs
   - "bg: N: no such job" for non-existent job IDs
   - "bg: job N already in background" for already running jobs

4. Integration:
   - Added to builtins table in builtins.c
   - Added declaration to builtins.h
   - Properly integrated with job tracking system
